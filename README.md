# 🎓 UBIT Student Hub Portal

A modern Flutter-based student portal designed to centralize academic, marketplace, and student services into one seamless experience.

---

## 🚀 Overview

UBIT Student Hub Portal is a mobile application that enables students to:

* 🔐 Securely log in to their accounts
* 🛒 Explore and interact with a student marketplace
* 📚 Browse and register for academic courses
* 📰 Stay updated with campus news and events
* 👤 Manage their personal profile

The app is designed with a clean UI/UX, ensuring smooth navigation and accessibility for students.

---

## 🧩 Features

### 🔑 Authentication

* User login system
* Form validation and session handling

### 🏠 Dashboard

* Overview of student activity
* GPA display and quick insights
* Campus news feed

### 🛒 Marketplace

* Browse student-relevant products
* Categorized listings (Books, Electronics, etc.)
* Product cards with pricing and details

### 📚 Course Management

* View available courses
* Register for courses
* Track academic selections

### 👤 User Profile

* Edit personal information
* Track profile completion
* Academic details management

### ⚙️ Navigation & Interaction

* Smooth bottom navigation
* Clean and responsive UI
* Logout confirmation modal
